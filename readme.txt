Simplifying fractions calculator to reduce fractions. Simplify proper and improper fractions and convert improper fractions to mixed numbers.

[https://www.calculator.io/simplifying-fractions-calculator/](https://www.calculator.io/simplifying-fractions-calculator/)

== Usage ==

Add the Simplifying Fractions Calculator shortcode to your page, post or sidebar:

`[ci_simplifying_fractions_calculator]`

Add the following code to your template where you would like the Simplifying Fractions Calculator to appear:

`<?php display_ci_simplifying_fractions_calculator(); ?>`

== Installation ==

1. Upload the Simplifying Fractions Calculator /ci_simplifying_fractions_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Simplifying Fractions Calculator plugin through the "Plugins" menu in WordPress.
